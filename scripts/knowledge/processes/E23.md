# Deep Knowledge: Reducción de ausencias a citas

## Lógica Técnica
Flujo de confirmación activa para optimizar la ocupación de la agenda.
1.  **Monitorización de Calendario**: n8n revisa las citas próximas para el día siguiente o las próximas 48 horas.
2.  **Envío de Recordatorio (tu canal de comunicación preferido)**: Envía un mensaje automático solicitando una confirmación activa (ej. "Pulsa 1 para confirmar o 2 para reprogramar").
3.  **Procesamiento de Respuesta**:
    -   **Confirmación**: Marca la cita como "Confirmada" en el calendario/CRM.
    -   **Reprogramación**: Envía automáticamente el enlace del Asistente de Reservas (E18) para elegir un nuevo hueco.
4.  **Alerta de Vacante**: Si el usuario cancela, avisa al equipo o dispara una promoción de "última hora" para ocupar el hueco libre.

## Beneficios y ROI
-   **Reducción Drástica de No-Shows**: Disminuye las ausencias imprevistas hasta en un 70%.
-   **Optimización de Ingresos**: Permite rellenar huecos cancelados con antelación suficiente.
-   **Mejora de la Experiencia**: El cliente agradece el recordatorio y la facilidad para cambiar la cita si le ha surgido algo.

## Casos de Uso
-   **Centros de Salud y Estética**: Donde una hora vacía supone un coste directo de personal.
-   **Talleres Mecánicos**: Gestión de citas de recepción de vehículos.
-   **Consultoría**: Confirmación de sesiones de asesoramiento de alto valor.

## Requisitos Previos
-   Uso de una agenda digital sincronizada.
-   Política de cancelaciones clara compartida con el cliente.

## Preguntas Frecuentes
-   **¿Qué pasa si no responden al mensaje?** El sistema puede intentar un segundo recordatorio más cerca de la hora o simplemente dejar la cita como "Pendiente" para que el equipo humano decida si llamar.
-   **¿Funciona con diferentes idiomas?** Sí, el bot puede detectar el idioma del cliente o usar el definido en su ficha.
