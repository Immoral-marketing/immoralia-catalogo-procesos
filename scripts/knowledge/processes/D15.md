# Deep Knowledge: Liquidación y facturación de freelance

## Contexto
Este proceso está diseñado exclusivamente para la **gestión de pagos y facturación de colaboradores externos, freelance o proveedores de servicios**. Se trata de un flujo de **egresos (gastos) y liquidaciones internas**, no de facturación a clientes finales.

## Lógica Técnica
Gestión de liquidaciones para colaboradores externos mediante el cruce de registros de tiempo y tarifas.
1.  **Captura de Tiempos**: n8n extrae las horas imputadas por cada freelance en herramientas como tu herramienta de gestión de tareas y tiempos, tu herramienta de gestión de tareas y tiempos o tu herramienta de gestión de tareas y tiempos.
2.  **Mapeo de Tarifas**: Cruza el ID del colaborador con una base de datos (Goole Sheets o Airtable) donde se definen los precios por hora acordados.
3.  **Cálculo e Impuestos**: Calcula el importe bruto y aplica las retenciones de IRPF correspondientes según el perfil del colaborador.
4.  **Generación de Factura/Gasto**: Crea automáticamente la factura de gasto en el ERP para que el freelance solo tenga que validarla o genera un aviso de "liquidación pendiente".

## Beneficios y ROI
-   **Transparencia**: Elimina disputas sobre el número de horas trabajadas.
-   **Ahorro Administrativo**: Ahorra horas de gestión manual cruzando Excels y mensajes para saber cuánto pagar a cada colaborador.
-   **Puntualidad en Pagos**: Los freelance cobran antes ya que el proceso de validación es mucho más rápido.

## Casos de Uso
-   **Agencias de Marketing**: Gestión de copywriters, diseñadores o desarrolladores externos.
-   **Empresas de Eventos**: Control de personal contratado por horas.

## Requisitos Previos
-   Uso de una herramienta de tracking por parte de los freelance.
-   Base de datos de tarifas actualizada.

## Preguntas Frecuentes
-   **¿Qué pasa si el freelance factura por proyecto?** El sistema puede configurarse para detectar hitos finalizados en lugar de horas.
-   **¿Detecta si las horas no están aprobadas?** Sí, se puede añadir un filtro para que solo facture las horas que han sido marcadas como "Aprobadas" por un PM.
