# Deep Knowledge: Presupuestos automáticos (A3)

## Contexto
Este proceso es una herramienta de **ventas y captación de clientes**. Su objetivo es agilizar el envío de propuestas comerciales por los servicios que la empresa vende. **No tiene relación con presupuestos recibidos de proveedores o colaboradores externos.**

## Lógica Técnica
Generación dinámica de documentos comerciales basados en datos estructurados.
1.  **Captura**: Datos de entrada desde un formulario (tu herramienta de formularios/tu herramienta de formularios) o una fila en Google Sheets.
2.  **Mapeo**: n8n cruza los servicios seleccionados con una base de datos de precios/tarifas.
3.  **Documentación**: Uso de la API de tu ERP/software de gestión para crear el presupuesto con todos los conceptos detallados.
4.  **Flujo**: El presupuesto queda creado en tu ERP/software de gestión listo para ser enviado o se envía automáticamente si se cumplen ciertas condiciones.

## Beneficios y ROI
-   **Velocidad de Respuesta**: Los leads reciben el presupuesto en minutos, aumentando el ratio de conversión.
-   **Uniformidad**: Todos los presupuestos siguen la misma estructura y diseño corporativo.
-   **Sin Errores**: Precios siempre actualizados desde una fuente centralizada.

## Casos de Uso
-   **Consultorías**: Servicios paquetizados con precios fijos.
-   **Agencias de Eventos**: Desglose rápido de partidas estándar.

## Requisitos Previos
-   Base de datos de servicios/productos configurada en el ERP.
-   Formulario de captura de datos o estructura de Sheets clara.

## Preguntas Frecuentes
-   **¿Puedo añadir productos personalizados?** Sí, el sistema puede tener una opción de "Otros" para añadir texto libre que luego un comercial revisa.
-   **¿Envía el PDF al cliente?** Se puede configurar, aunque recomendamos que pase por una revisión manual antes de salir.
