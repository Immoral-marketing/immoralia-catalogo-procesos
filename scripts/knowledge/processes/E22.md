# Deep Knowledge: Atención automática por tu canal de comunicación preferido

## Lógica Técnica
Gestión automatizada de la presencia social para mejorar la interactividad sin esfuerzo humano constante.
1.  **Entrada de DMs**: n8n se conecta con la API de tu canal de comunicación preferido Messaging (via Facebook Developer portal).
2.  **Análisis Sugestivo (IA)**: Procesa el mensaje del usuario para detectar si es una duda genérica, un comentario sobre una publicación o un interés de compra.
3.  **Respuesta Instantánea**: Si la consulta es estándar, la IA responde citando la información del catálogo o resolviendo la duda con el tono adecuado.
4.  **Derivación**: Si el usuario muestra una intención clara de compra o solicita atención humana, se notifica al equipo por el canal de comunicación interna.

## Beneficios y ROI
-   **Tasa de Respuesta 100%**: Evita perder el badge de "responde rápido" de tu canal de comunicación preferido.
-   **Captura de Leads en Social**: Convierte seguidores curiosos en leads registrados en el CRM de forma fluida.
-   **Ahorro de CM**: Libera al Community Manager de responder cientos de mensajes de "precio" o "info" cada día.

## Casos de Uso
-   **Retail y Moda**: Consultas sobre stock o tallajes directamente por DM.
-   **Agencias Creativas**: Recepción de solicitudes de presupuesto por mensaje directo.

## Requisitos Previos
-   Cuenta de tu canal de comunicación preferido vinculada a una Fanpage de Facebook.
-   Acceso a la API de Messenger para tu canal de comunicación preferido.

## Preguntas Frecuentes
-   **¿Puede responder a comentarios en posts?** Sí, se puede configurar para que si alguien comenta una palabra clave (ej. "INFO"), el bot le envíe automáticamente un DM con los detalles.
-   **¿Funciona con stories?** Sí, puede responder de forma automática a las reacciones o menciones en stories.
